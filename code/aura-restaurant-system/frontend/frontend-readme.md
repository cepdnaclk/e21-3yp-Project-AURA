# Frontend - AURA Restaurant System

This project is built using React and Vite. Follow the instructions below to set up and run the frontend application.

## Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

## Getting Started

1. **Navigate to the frontend directory:**
   ```bash
   cd code/aura-restaurant-system/frontend
   ```

2. **Install dependencies:**
   Using npm:
   ```bash
   npm install
   ```
   *Or using yarn:*
   ```bash
   yarn install
   ```

3. **Run the development server:**
   Using npm:
   ```bash
   npm run dev
   ```
   *Or using yarn:*
   ```bash
   yarn dev
   ```

4. **Open your browser:**
   The application will generally be available at `http://localhost:5173` (or the URL provided in your terminal).

## Building for Production

To build the project for production, run:
```bash
npm run build
```
This will compile the application into the `dist` folder.
